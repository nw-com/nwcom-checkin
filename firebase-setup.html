🔧 Firebase 設定助手

讓您的西北勤務管理系統快速上線！

步驟 1：創建 Firebase 專案

1. 前往 Firebase Console

2. 點擊「新增專案」或「建立專案」

3. 輸入專案名稱（例如：nwcom-checkin）

4. 啟用 Google Analytics（可選）

5. 點擊「建立專案」

步驟 2：啟用認證服務

1. 在左側選單點擊「Authentication」

2. 點擊「開始使用」

3. 選擇「電子郵件/密碼」登入方式

4. 啟用「電子郵件/密碼」並儲存

步驟 3：啟用 Firestore Database

1. 在左側選單點擊「Firestore Database」

2. 點擊「建立資料庫」

3. 選擇「以測試模式啟動」

4. 選擇資料庫位置（建議選擇最近的區域）

5. 點擊「啟用」

步驟 4：獲取 Firebase 配置

1. 點擊左上角的齒輪圖示 →「專案設定」

2. 在「一般」標籤下，找到「您的應用程式」區塊

3. 點擊「」圖示（網頁應用程式）

4. 輸入應用程式名稱（例如：nwcom-checkin）

5. 點擊「註冊應用程式」

6. 複製下方的配置代碼

步驟 5：更新您的系統

1. 將生成的配置貼到 js/firebase-config.js

2. 儲存檔案

3. 重新載入您的系統

步驟 6：設定資料庫規則（重要）

1. 前往 Firestore Database

2. 點擊「規則」標籤

3. 貼上以下安全規則：

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if request.auth != null; } } }

4. 點擊「發布」

🎯 恭喜！ 完成以上步驟後，您的西北勤務管理系統就可以正式運作了！
